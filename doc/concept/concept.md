
# ged 

geoserver geometry editor

## concept

* geoserver geometry editor
* supposed to be a geometry editor for geoserver layers
* try ui only, using geoserver and wms/wfs api only
* use wfs transactions to edit layers
* user can
    - select layers form list
    - show layer son map
    - edit layer geometry
    - create a new layer
